# Hair Salon Booking System

A modern and responsive salon booking application built with React that allows customers to schedule appointments quickly and easily. The application provides a simple booking workflow and uses browser Local Storage to persist appointment data without requiring a backend service.

## Live Demo

https://hair-salon-booking-system.vercel.app/

---

## Overview

The Hair Salon Booking System was developed to demonstrate how appointment scheduling can be implemented using a frontend-only architecture.

The application allows users to select salon services, choose available booking slots, and manage appointments through an intuitive interface. All booking information is stored locally within the browser using Local Storage, making the application lightweight and easy to deploy.

This project focuses on user experience, responsive design, and state management while showcasing practical frontend development skills.

## Installation

Clone the repository:

```bash
git clone https://github.com/Daniell-fejiro/hair-salon-booking-system.git
```

Navigate into the project:

```bash
cd hair-salon-booking-system
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open your browser:

```bash
http://localhost:5173
```
